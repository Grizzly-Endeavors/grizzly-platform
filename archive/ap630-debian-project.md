# AP630 Debian Router Project — Moved

The AP630 Debian port (custom kernel, DTS, driver patches, rootfs builder, RDP reverse-engineering notes, and stock-firmware restore tooling) has been extracted into its own public repository:

**<https://github.com/Grizzly-Endeavors/ap630-debian>**

The project is archived — see [ADR-011](decisions/011-ap630-restored-to-stock-wifi-ap.md) for why. The AP630 now runs stock HiveOS and serves as a WiFi access point; it is not active infrastructure in this repo.
